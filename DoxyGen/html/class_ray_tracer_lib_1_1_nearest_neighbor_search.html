performs a non-recursive K nearest neighbors search in a flat list.
Algorithm :

  • First time we arrive on a node, we go to its "best" child (the one that is on the same size as the input point)
  • Second time we arrive on a node, we go to its "worst" child if we need to (see knn search in kd trees)
  • Third time we arrive on a node, we go back to its parent
Parameters
pixelIndexThe index of the pixel for which we are running this search
pointThe point for which we want to search neighbors
treeDepthThe depth of the "tree" we are searching
photonsTreeBufferThe GPU buffer containing the tree data
photonsNeighborsBufferThe GPU buffer that will contain the neighbors found
photonsSearchStateBufferThe GPU buffer that will contain the state of the search (the number of visits at each depth)
Returns
The number of neighbors found
