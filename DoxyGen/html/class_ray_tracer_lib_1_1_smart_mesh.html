Detailed Description

A mesh structure that recursively stores triangles in a tree structure that makes collision detection very fast.
⚠️ The triangles will be moved and scaled such that their bounding box is centered around 0 and has a max size of 1 in any direction.

  • compute the bounding box of the given triangles
  • split this bounding box in 8 smaller ones if it contains too many triangles
  • allocate each triangle to the bounding box that contains it
  • repeat recursively
